¿Qué es Cloud Computing?

Cloud Computing es un modelo de entrega de servicios de computación a través de internet, donde recursos como servidores, almacenamiento, bases de datos, redes, software y análisis se proporcionan de manera escalable y elástica, sin depender de hardware local.

Según el NIST, es un modelo que permite acceso ubicuo y bajo demanda a recursos configurables con mínimo esfuerzo de gestión. Pay-as-you-go: pagas solo por lo que usas.

Ejemplo: en lugar de comprar servidores, "alquilas" capacidad de AWS, Azure o GCP.

Historia del Cloud Computing

Evolución desde los mainframes hasta la nube moderna

De los mainframes de los 60s a la nube actual

Año Hito Principal Detalles
1950s-1960s Orígenes en mainframes J.C.R. Licklider propone red intergaláctica (precursora de internet).
1969 ARPANET Primera red que permite compartir recursos digitales.
1970s Virtualización IBM lanza VM. Nace el símbolo de "nube".
2006 AWS lanza S3 y EC2 Populariza IaaS. Google lanza Docs (SaaS).
2008 Google App Engine Primer PaaS. Microsoft anuncia Azure.
2020s Boom AI y Hybrid Integración con edge computing, IA y crecimiento del 30% anual.
